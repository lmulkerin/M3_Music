# Description:
This repo contains a Sonic Pi file containing background music for an educational song about recursion. This song is intended to follow a typical intro to computer science curriculum and help students study for an exam. While composing, I wanted the song to sort of resemble a circus feeling, because I thought it would bring a stress-free lightheartedness to student studying while embodying just a bit of the chaos of this time of year. The simple rhythms, key, repetition of note progressions, and overall song organization are an attempt at this feeling.
